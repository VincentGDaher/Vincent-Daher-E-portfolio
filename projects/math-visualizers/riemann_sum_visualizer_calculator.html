Method Formula idea Approximation Error

1. Cut

The interval is divided into n equal parts. Each part has width Δx.

2. Choose height

Left, right, or midpoint methods choose a different height in each subinterval.

3. Add rectangles

Each rectangle contributes height × width. The sum approximates area.

4. Take the limit

As n → ∞, the rectangles become thinner and the sum approaches the integral.
